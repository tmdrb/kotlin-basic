# 클로저, 람다 비교

## 클로저

보통 함수는 외부에서 인자를 받아와서 사용하지만 클로저는 외부 영역에 있는 변수를 함수내로 바인딩해서 사용하는 방식이다.

특이한점으로는 외부의 함수가 종료되어도 값이 사라지지 않는다.



# kotlin 표준 라이브러리 함수

## let

실제 let 함수 코드

fun <T,R> T.let(block:(T)->R): R

T를 받아서 R 로 리턴해준다.

let을 사용하게 되면 null 체크를 하지 않고 프로퍼티의 값을 바꿀수 있다.
